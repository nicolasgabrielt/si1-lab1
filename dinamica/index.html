Porque me matricular em SI?

A web se tornou uma ferramenta imprescindível para todos no século que estamos. Além disso, ela tem transformado a forma de comunicação e de negócios , um exemplo disso é que hoje 90% das grandes empresas tem sites e sua maior forma de marketing e de vendas é na internet. Visto todos esses benefícios da internet, porque não dominarmos essa ferramenta tão importante, e através disso nos tornarmos profissionais mais cobiçados pelo mercado de trabalho. E em SI 1 você terá a oportunidade de aprender e evoluir nessa ferramenta .

SI é uma das cadeiras que tem projeto, então é uma oportunidade de você criar um produto(site) de sua autoria ou dar progresso a uma ideia que você já tinha a muito tempo. Cadeiras com projeto nos deixam mais motivados pelo curso, pois é muito gratificante ver algo de sua autoria pronto e servindo pra alguma coisa.

Play! Framework

O Play framework é famoso por facilitar, e muito, a vida do desenvolvedor que precisa criar uma aplicação MVC de forma fácil e rápida. Os recursos que o framework disponibiliza impressionam pela simplicidade e rápida curva de aprendizado. Além de possuir vários recursos embutidos, o playframework possui uma boa lista de módulos que podem ser “plugados” à aplicação, disponibilizando recursos mais avançados, como, por exemplo, Web Services, integrações com redes sociais Facebook e Twitter, testes automatizados, entre outros.

GitHub

GitHub é um Serviço de Web Hosting Compartilhado para projetos que usam o controle de versionamento Git. É escrito em Ruby on Rails pelos desenvolvedores da Logical Awesome (Chris Wanstrath, PJ Hyett e Tom Preston - Wernder). O GitHub possui planos comerciais e gratuitos para projetos de código aberto. Este site possui funcionalidades de uma rede social como feeds, followers, wiki e um gráfico que mostra como os desenvolvedores trabalham as versões de seus repositórios.

GIT

Git é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase em velocidade. O Git foi inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento do kernel Linux, mas foi adotado por muitos outros projetos. Cada diretório de trabalho do Git é um repositório com um histórico completo e habilidade total de acompanhamento das revisões, não dependente de acesso a uma rede ou a um servidor central.

IntelliJ IDEA

IntelliJ IDEA permite escrita de código Java sem complicações. Ele pratica uma abordagem não-intrusiva intuitiva para ajudá-lo a escrever, depurar, refatorar, testar e aprender o seu código. Graças à sua profunda compreensão das linguagens e tecnologias, IntelliJ IDEA oferece um segundo par de mãos quando você precisar deles.

Boostrap

O Bootstrap tem como objetivo tornar mais fácil o desenvolvimento de interface (front-end) para páginas web. Ele disponibiliza padrões para os elementos HTML mais usados, além de elementos personalizados com o usado de elementos e classes css padrões.

WebStorm

WebStorm é uma IDE leve e poderosa, perfeitamente equipada para o complexo desenvolvimento do lado do cliente e desenvolvimento do lado do servidor com Node.js.
